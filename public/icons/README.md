# Track Records icons

Icons from **Figma** ([Apee 2025 – node 2027-758](https://www.figma.com/design/b2PtdH5HtmnppuibOcsS50/Apee-2025?node-id=2027-758)).

| File | Usage |
|------|--------|
| `Icon-caret-down.svg` | Dropdown chevron for "Companies" and "Types" (rotates when open) |
| `Icon-rocket.svg` | Featured list row icon (1st, 6th) |
| `Icon-tada.svg` | Featured list row icon (2nd, 7th) |
| `Icon-component.svg` | Featured list row icon (3rd, 8th) |
| `Icon-star-shiny.svg` | Featured list row icon (4th, 9th) |
| `Icon-signature.svg` | Featured list row icon (5th, 10th) |

The Featured section cycles through the five icons in order for each row.
